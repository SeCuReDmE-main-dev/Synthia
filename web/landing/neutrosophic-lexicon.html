How the neutrosophic lexicon becomes code

Foundation T/I/F

Formal profiles are preserved, then normalized into bounded operational `TIF` values for scoring and review.

Set membership

`T` as membership, `I` as indeterminacy, and `F` as non-membership with complete, incomplete, and paraconsistent diagnostics.

Logic propositions

Relative/absolute, incomplete, paraconsistent, paradoxist, IFL-compatible, and general neutrosophic logic profiles.

Single-valued operations

Bounded union, intersection, difference, truth-favorite, and falsity-favorite operators.

Probability

Event triples, complement summaries, sample-space diagnostics, and probability-to-I_lexicon projection.

Statistics

Known/interval/unknown summaries, indeterminacy load, candidate distributions, and review-risk routing.

Random variables

Symbolic PDF/PMF/CDF metadata, expected value, variance, standard deviation, and stochastic I_lexicon projection.

Plithogenic methods

Dominant attribute contradiction, many-variable proposition truth, and cumulative truth classification as system-level indeterminacy.

Boundary

Candidate routing, not final authority

Synthia uses public NSS sources to structure safe software behavior. Generated classifications are source-linked candidates for review, not final mathematical, scientific, medical, legal, or nomenclatural authority.
