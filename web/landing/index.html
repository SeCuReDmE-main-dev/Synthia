What Synthia is now

A research core for classification memory

Synthia is not yet a large model or production platform. Its current value is architectural: it gives the project a clean public skeleton for lexicon-aware classification, source evidence, uncertainty, and taxonomy memory repair.

01

I_lexicon nodes

Concepts carry domain, definition, source IDs, T/I/F values, and lexicon state.

02

Context switching

A task can move between biology, taxonomy, PhyloCode, math, and AI governance without losing evidence.

03

Typed bridges

Terms connect through synonymy, redescription, rank shift, analogy, contradiction, and memory repair.

04

Plithogenic profiles

Multi-attribute contradiction and T/I/F scoring remain visible instead of collapsing into a single confidence number.

05

Taxonomy memory

Species names can be treated as memory objects linking descriptions, redescriptions, synonymy, ecology, and review.

06

Private boundary

Evidence ledgers, drafts, and private correspondence stay outside the public repository.

I_lexicon engine

A classification system that knows which language it is using

The goal is not only to classify a document. The goal is to preserve the lexicon behind the classification: biology terms, PhyloCode terms, neutrosophic math terms, governance terms, and the bridges between them.

Scientific collaboration

Jean-Sebastien Beaulieu leads Synthia's software architecture and implementation. Prof. Hector Fernando Aguilar is credited as scientific co-conceptor for the taxonomy, systematics, PhyloCode/classification direction, redescription-memory framing, and expert-review boundary.

This credit recognizes scientific conception and field-taxonomy guidance. It does not transfer software authorship, formal nomenclatural authority, or autonomous scientific decision-making to Synthia.

Boundary

A serious research instrument begins with disciplined claims

Synthia can model ambiguity, traceability, and taxonomic memory repair. Formal nomenclatural authority remains human and code-governed. Scientific, medical, conservation, and field decisions require expert validation.
