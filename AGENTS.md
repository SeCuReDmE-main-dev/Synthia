# AGENTS.md

This is an official SecuredMe school-tool repository.

Before changing code or documentation, preserve the education-first boundary:

- Official AI-assisted school route: Codex/OpenAI or Antigravity/Gemini only.
- Do not add Ollama Cloud, uncensored local AI, or unknown agent providers as official classroom routes.
- Do not rewrite core mechanisms casually; make the smallest coherent change and use pull requests for official support.
- If a user wants a different mechanism, build a separate local fork, adapter, or plugin instead of changing the maintained school mechanism.
- Do not introduce attack, fraud, theft, bypass, or abuse workflows.
- Preserve this repo's scientific and human-review authority boundaries.

The maintainer does not support private modified copies that bypass review.
