## Submission decision

Synthia should be presented as a free, source-available educational research demo built
with Codex, not as an OpenAI-powered production service. Its differentiator is
the **living taxonomy trace**: public evidence is transformed into a versioned,
reviewable candidate memory while sources, uncertainty, and human scientific
authority stay visible.

The hosted Trace Lab makes that proposition understandable in under a minute.
The Python CLI proves that the public JSON is generated by the repository rather
than hand-authored for the website.

## Demonstration script

1. Open the hosted Trace Lab and identify the schema
   `synthia.showcase.trace.v1`.
2. Follow the 1828 → 2012 → 2026 evidence timeline.
3. Contrast the restored memory layers with the explicitly unresolved layers.
4. Open a public source from the registry.
5. Switch from English to French, Spanish, or Italian.
6. Show the authority boundary and the invariant
   `I -> I_system^S -> H_lex -> G_lex -> I_lexicon`.
7. Copy the local CLI command and regenerate the same JSON from the repository.

## Architecture

```text
public sources
      |
      v
curated showcase builder -- fail-closed privacy/authority checks
      |
      v
synthia.showcase.trace.v1 JSON
      |                         \
      v                          v
Python CLI proof            static Trace Lab
                               |
                               v
                    four languages + human review
```

The showcase package is deliberately separated from Synthia's private evidence
ledger. It cannot read private Drive or correspondence identifiers. It also does
not introduce the separate fractal hierarchy into the public I_lexicon chain.

## Scientific source boundary

- Formal authority displayed by the case: Lesson, 1828.
- Critical redescription metadata: Aguilar, H. F. & Aguilar H., R. F. (2012),
  *Revista de Ecología Latinoamericana* 17(3):53-61.
- Current peer-reviewed context: [Taxonomy of Aburria Reichenbach, 1853
  (Aves, Galliformes, Cracidae) based on morphological
  characters](https://doi.org/10.11606/1807-0205/2026.66.014), 2026.
- Current registry context: [NCBI Taxonomy Browser, Taxonomy ID
  125058](https://www.ncbi.nlm.nih.gov/Taxonomy/Browser/wwwtax.cgi?id=125058&mode=info).

These sources support a review trace. They do not turn Synthia into a formal
nomenclatural authority.

## Why this can stand out

Many AI demos optimize for a fluent final answer. Synthia makes the memory
construction visible: what was restored, what remains unresolved, which source
supports each step, which lexicon governs the classification, and where the AI
must stop. That combination of scientific humility, deterministic developer
tooling, multilingual explanation, and a no-key public demo gives the project a
clear identity without overstating its maturity.
